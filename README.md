# embedded-system-heart-rate

## Summary
This project uses Wokwi Custom Chip API to build a simulated SEN-11574 Pulse Sensor, connected to an ESP32. 

## Features
Wi-Fi connectivity.
EPS32 controller integration.
LED integration.
Pulse reading simulation.


## Dependencies
Wokwi Chips API
Wokwi WiFi API
HTTP Client API
